#  SQL Projects

This repo contains two beginner SQL projects using SQLite.

##  Projects

- **[Personal Expense Tracker](personal-expense-tracker-sql/)**- Track personal expenses, calculate totals, and analyse spending by category.
-  **[Employee Sales Tracker](employee-sales/)**- Analyse employee sales data, totals per employee, and average per department.

  Each folder has its own README with setup instructions, SQL queries, and screenshots.
